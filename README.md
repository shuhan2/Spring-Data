1.Spring 中 Bean的Scope有哪些？它们分别代表什么含义？<br>

| Scope   | 含义|
| ------------- |:-------------:| 
|    单例(singleton)       |     容器将单个bean范围限定为为单个对象实例       | 
|     原型(prototype)      |      容器将单个bean限定为仁矣对象的实例        |   
| 请求(request) | 0L      |   
| 会议(session) | 将单个bean定义范围限定为HTTP的生命周期session|
| 全局会议(global session	) | 将单个bean定义范围限定为全局HTTP的生命周期session|
